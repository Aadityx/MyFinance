# 🚀 ZION – Expense Tracker Web App

## 📌 Overview

This project was developed during **ZION**, a 3-hour coding event. The goal was to select a problem statement and build a fully functional web solution within a limited timeframe.

We built a **smart expense tracking web application** that helps users monitor and analyze their spending habits in real-time.

---

## 💡 Features

* Add expenses with details (name, category, amount)
* Classify expenses as **Impulse** or **Planned**
* Visual representation using **donut charts**
* Track **total expenditure** and category-wise distribution
* View **transaction history**

---

## 🧩 Pages Included

1. **Login Page** – User authentication entry point
2. **Registration Page** – New user sign-up
3. **Dashboard** – Overview with insights and charts
4. **Add Expense** – Input expense details
5. **Transaction History** – View past records

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** JavaScript
* **Database:** Firebase

---

## 📊 How It Works

1. User enters expense details
2. Data is stored in the backend (Firebase)
3. Data is processed and categorized
4. Insights are generated and displayed using charts
5. Users can track and analyze their spending behavior
